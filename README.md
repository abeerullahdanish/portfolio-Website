## About This Project

This is my personal portfolio website where I showcase my work as a Data Scientist.
I built it from scratch using HTML, CSS and JavaScript.
The goal was simple — have one clean place where people can see who I am,
what I know, and what I have built so far.

---

## What's Inside

- A hero section with my name and a typewriter that shows my roles
- An about section with the tools and technologies I work with
- My education and freelance experience
- A services section showing what I can do for clients
- 6 of my Data Science projects with descriptions and GitHub links
- A contact section with a form and my social links
